# linux_service_exporter
Prometheus linux service exporter.

### require
* go version : 1.9.0+
* prometheus : 2.0.0+
* os         : linux

### install
```bash
$ go get tomoncle/linux_service_exporter
$ cd $GO_PATH/src/github.com/tomoncle/linux_service_exporter
$ go build linux_service_exporter.go
```
